Scylla 中文文档
==================================

.. toctree::
:maxdepth: 2
       :caption: Contents:

    Scylla 是一款高质量的免费代理 IP 池工具，有着如下特性：

    -  自动化的代理 IP 爬取与验证
    -  易用的 JSON API
    -  简单但美观的，基于 web 的用户界面（例如，代理的地理分布）
    -  最少仅用\ **一条命令**\ 即可启动
-  简明直接的编程 API（将在 1.1 版本中加入）
-  （可选）无头浏览器（headless browser crawling）爬虫
