===

A minimal proxy ip crawler, only supports Python 3.6.



## Get started

### Install directly via pip

```bash
pip install scylla
scylla # Run the cralwer and web server for JSON API
```

### Use from source

```bash
git clone https://github.com/imWildCat/scylla.git
cd scylla
python -m scylla
```
